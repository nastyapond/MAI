# lab03 - Классификация и прогнозирование метеоданных
### Прудникова Анастасия 214

## О проекте
Это лабораторная работа по построению двухэтапного решения для метеоданных шести городов России. Проект включает:
- классификацию климатической зоны по краткосрочному временному ряду;
- прогноз температуры на 30 дней для каждого города;
- объединение этапов в единый пайплайн.

## Содержание
- `lab03.ipynb` - Jupyter Notebook с полным решением и аналитикой
- `requirements.txt` - зависимости Python
- `data/` - каталог с исходными данными

## Запуск
1. Перейти в папку `lab03`:
   ```bash
   cd lab03
   ```
2. Создать и активировать виртуальное окружение:
   ```bash
   python -m venv .venv
   .\.venv\Scripts\activate
   ```
3. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустить Jupyter Notebook и открыть `lab03.ipynb`:
   ```bash
   jupyter notebook lab03.ipynb
   ```
5. Выполнить ячейки последовательно


## Ход работы
2.1 EDA: визуальный анализ, сезонность, пропуски, аномалии, корреляции, стационарность

2.2 Инжиниринг признаков: временные признаки, циклические признаки, лаги, скользящие окна и динамика

2.3 Классификация: специализированная TSC-модель (ROCKET/DTW-kNN/TSF) и базовый RF для сравнения

2.4 Оценка классификации: accuracy, precision, recall, F1, confusion matrix, ROC-AUC

2.5 Прогнозирование: отдельные модели для климатических зон и городов

2.6 Оценка прогноза: MAE, RMSE, MAPE, WAPE, диагностика остатков

2.7 Пайплайн: классификация -> выбор модели -> прогноз

2.8 Документация: итоговые выводы и инструкции

## Результаты
- Классификация TSC-модели: **Accuracy 72.67%**.
- Базовый RF на агрегированных признаках: **Accuracy 82.80%**.
- Прогнозные модели зон:
  - `northwest` (RF) - MAE 3.45, RMSE 4.18.
  - `blacksea` (GBR) - MAE 2.93, RMSE 3.79.
  - `far_east` (ExtraTrees) - MAE 2.79, RMSE 3.62.
- Пайплайн правильно определил климатическую зону для всех шести городов в выборке.

## Вывод
Двухэтапный подходит для использования в данном случае, так как помогает адаптировать прогнозную модель под климатическую зону и улучшает качество прогноза в сравнении с универсальной моделью для всех городов.
